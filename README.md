# Project Repository

Welcome to this project repository! This workspace contains multiple data science and analysis projects using Python, NumPy, Pandas, and various data analysis techniques.

## Directory Structure

### 📚 Assignment/
Learning materials and practice workbooks for fundamental data science libraries.
- **NumPy_Student_Workbook_(1).ipynb** - NumPy concepts and operations
- **Pandas_Student_Workbook_(2).ipynb** - Pandas data manipulation and analysis

### 🚗 Major/
Road accidents analysis project for India (2022-2025).
- **India_Road_Accidents_2022_2025.csv** - Dataset containing road accident records
- **main project report.pbit** - Power BI interactive report and visualizations

### 🛒 minor/
Amazon data science project and analysis.
- **Amazon_DataScience.ipynb** - Analysis notebook for Amazon dataset
- **amazon.csv** - Amazon products/data dataset
- **matches.csv** - Additional matches dataset
- **min.ipynb** - Secondary analysis notebook

## Getting Started

### Prerequisites
- Python 3.x
- Jupyter Notebook
- Required libraries: NumPy, Pandas, etc.

### Installation
1. Clone or download this repository
2. Install required dependencies:
   ```bash
   pip install numpy pandas jupyter
   ```
3. Navigate to the desired project folder
4. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Projects Overview

### Assignment Projects
Practice notebooks covering fundamental data science concepts:

- Basic statistical analysis

### Major Project: India Road Accidents Analysis
Comprehensive analysis of road accidents across India from 2022-2025 with interactive visualizations in Power BI.

## Usage

Each Jupyter notebook can be run independently. Open the relevant `.ipynb` file in Jupyter and execute the cells to:
- Learn data science concepts
- Analyze datasets
- Generate insights and visualizations

## Notes

- Ensure all CSV files are in the same directory as their corresponding notebooks
- Power BI reports can be opened with Microsoft Power BI Desktop
- Update datasets as needed for your analysis
